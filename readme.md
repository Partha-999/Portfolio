A **sleek, interactive portfolio** built with **Next.js**, showcasing projects, skills, and 3D animations. Designed for performance, responsiveness, and a premium UX.

---

## 💻 Tech Stack

- **Next.js** – React framework for SSR & performance  
- **TypeScript** – Strong typing for safe, scalable code  
- **React** – Component-based UI library  
- **React Three Fiber** – 3D graphics in React using Three.js  
- **Drei** – Useful helpers for React Three Fiber  
- **Tailwind CSS** – Rapid, utility-first styling  
- **GSAP** – Smooth, advanced animations
